🌍 Environmental Pollution Dashboard – India

📊 Project Overview  
This project presents an interactive Environmental Pollution Analysis Dashboard built using Power BI. The goal is to analyze pollution levels across different states, districts, and pollutants in India to uncover trends, identify high-risk regions, and support data-driven insights.

🛠 Tools Used  
Power BI  
DAX (Data Analysis Expressions)  
Data Modeling  
Excel (Data Cleaning & Preparation)  

📁 Dataset Features  
State  
District  
Station  
Pollutant (CO, NO₂, PM10, PM2.5, SO₂, etc.)  
Min Pollution  
Max Pollution  
Avg Pollution  
Time Period  

📌 Key Insights  
Identified top polluted states and districts  
Analyzed pollutant-wise concentration levels  
Observed variation between minimum, average, and peak pollution  
Classified pollution into severity levels (Good → Critical)  
Highlighted dominant pollutants across regions  

📈 Dashboard Features  
KPI Cards (Avg Pollution, Peak Pollution, Total Records, Coverage)  
Pollutant-wise comparison (CO, NO₂, PM10, PM2.5, etc.)  
State-wise pollution analysis  
Heatmap (State × Pollutant intensity)  
Severity distribution (Donut chart)  
Time-based trend analysis  
Interactive filters (State, District, Date)  
Multi-page navigation (Overview | Analysis | Trends)
